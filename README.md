# Gestão de Contatos

## Instalação
1. Crie o banco de dados usando `banco.sql`.
2. Configure `php/db.php` com suas credenciais.
3. Coloque o projeto em um servidor local (XAMPP, WAMP).
4. Acesse `index.html`.

## Funcionalidades
- Cadastro, edição e remoção de contatos.
- Validações front-end e back-end.
- Paginação.
- Upload de foto.
- Filtro dinâmico.
